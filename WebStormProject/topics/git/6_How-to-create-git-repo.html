How to create GIT repository?

Posted by Arpit Jain on August 09, 2016

Don't forget to checkout the licensing/usage policy before trying out.

Create an account on Github

  1. Signup (new account) with your favorite github username and some email ID, which you already use. This email ID will be considered as github email ID.

Sign in to Github .

Create New repository

  1. Click on New repository .
  2. Provide repository details like Repository name , Description , public or private (public is free) etc. then click on Create repository .
  3. Ignore this screen for the time being, we will come back and learn more in upcoming tutorials.
