How to install GIT client (Bash & GUI)?

Posted by Arpit Jain on August 09, 2016

Don't forget to checkout the licensing/usage policy before trying out.

Download clients.

  1. Search for git client download on google.
  2. Download for Windows .
  3. Download will start, wait for download to finish.

Install clients. (git bash and git gui)

  1. Once client is downloaded, launch the setup. Read the license and click on Next > .
  2. Specify installation directory.
  3. Select all choices, like shortcuts, Windows explorer integration.
  4. Change if you want to alter the git shortcut placement in Start menu, Otherwise leave as default and proceed.
  5. Choose PATH related choice. Let the default option be selected and proceed.
  6. Let the default option be selected and proceed.
  7. Let the default option be selected and proceed.
  8. Let the default option be selected and proceed.
  9. Finally, installation will start now. Wait for sometime to finish the installation.
  10. Click on "Finish".

Launch git bash and git gui

  1. Go to Start Menu -> All Programs -> Git -> Git Bash to start command line client.
  2. Go to Start Menu -> All Programs -> Git -> Git GUI to start GUI client.
